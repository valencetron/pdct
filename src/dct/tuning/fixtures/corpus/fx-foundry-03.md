This note records session 3 of work on a small sculpture foundry's casting practice. The main focus was crucible preheat and how it interacts with sprue venting. We measured the effect carefully and wrote down every reading in the shared log. The results were consistent with what the previous session predicted, which builds confidence in the method.

A second thread involved patina bath. The key decision was to adjust the procedure gradually rather than all at once, because sudden changes made earlier comparisons useless. We agreed that the next session should revisit crucible preheat with the improved tooling and confirm the numbers hold. Several small anomalies were noted and each one was given a follow-up owner.

Finally, the team discussed how sprue venting relates to the wider program. The consensus was that documentation quality matters as much as the raw data, so this distillation captures the reasoning, not just the outcome. Anyone picking this up later should start from the summary above and then check the follow-up list before changing anything.
