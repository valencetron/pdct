This note records session 3 of work on a volunteer marine tidepool monitoring program. The main focus was spring tide window and how it interacts with transect lines. We measured the effect carefully and wrote down every reading in the shared log. The results were consistent with what the previous session predicted, which builds confidence in the method.

A second thread involved salinity logger. The key decision was to adjust the procedure gradually rather than all at once, because sudden changes made earlier comparisons useless. We agreed that the next session should revisit spring tide window with the improved tooling and confirm the numbers hold. Several small anomalies were noted and each one was given a follow-up owner.

Finally, the team discussed how transect lines relates to the wider program. The consensus was that documentation quality matters as much as the raw data, so this distillation captures the reasoning, not just the outcome. Anyone picking this up later should start from the summary above and then check the follow-up list before changing anything.
