This note records session 4 of work on a small sculpture foundry's casting practice. The main focus was sprue venting and how it interacts with patina bath. We measured the effect carefully and wrote down every reading in the shared log. The results were consistent with what the previous session predicted, which builds confidence in the method.

A second thread involved kiln firing curve. The key decision was to adjust the procedure gradually rather than all at once, because sudden changes made earlier comparisons useless. We agreed that the next session should revisit sprue venting with the improved tooling and confirm the numbers hold. Several small anomalies were noted and each one was given a follow-up owner.

Finally, the team discussed how patina bath relates to the wider program. The consensus was that documentation quality matters as much as the raw data, so this distillation captures the reasoning, not just the outcome. Anyone picking this up later should start from the summary above and then check the follow-up list before changing anything.
